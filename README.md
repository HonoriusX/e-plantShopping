# e-plantShopping
Paradise Nursery Online Plant Shop Shopping Cart Application

## Project Description
A React + Redux Toolkit e-commerce application for Paradise Nursery, featuring product browsing, cart management, real-time quantity updates, and total cost calculation.

## Core Features
- Landing page with background image and "Get Started" button
- Global navigation bar with Home/Plants/About Us/Cart links
- 3 plant categories (6 plants each) with card-based display
- Add-to-cart functionality with disabled button state after adding
- Cart item quantity adjustment, deletion, and real-time subtotal/total calculation
- Empty cart friendly message and checkout placeholder
